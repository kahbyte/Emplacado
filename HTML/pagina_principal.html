Sobre o Projeto

O objetivo do projeto é classificar imagens, com o auxílio de bibliotecas como Tensorflow e Keras para técnicas de deep, Pillow para manipulação de imagens e o Discord para a criação do bot. O projeto foi desenvolvido com base no GTSRB, German Traffic Sign Recognition Benchmark, manipulado através do Google Colab para mostrar não apenas as saídas, mas também gráficos de performance.

Linguagens e Softwares Utilizados
